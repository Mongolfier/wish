# Wish UI Kit

Welcome to the **Wish** UI Kit — a library of reusable React components.

## Structure

<table>
  <thead>
    <tr>
      <th>Category</th>
      <th>Description</th>
    </tr>
  </thead>
  <tbody>
    <tr><td><strong>Tokens</strong></td><td>Design tokens: colors, fonts, spacing</td></tr>
    <tr><td><strong>Assets</strong></td><td>Icons and other static resources</td></tr>
    <tr><td><strong>Core</strong></td><td>Base components (Icon, Button, etc.)</td></tr>
    <tr><td><strong>Elements</strong></td><td>Simple UI elements</td></tr>
    <tr><td><strong>Molecules</strong></td><td>Compound components built from multiple elements</td></tr>
    <tr><td><strong>Organisms</strong></td><td>Large interface blocks (Header, Footer, etc.)</td></tr>
  </tbody>
</table>

## Quick start

```bash
pnpm add @wish/ui-kit
```

```tsx
import { Icon } from '@wish/ui-kit';

export const App = () => (
  <Icon category="actions" name="copy" width={24} height={24} />
);
```

## Documentation in MDX

Each component can have its own MDX page next to its stories file.
This allows you to write detailed documentation with examples, guidelines, and live previews.
