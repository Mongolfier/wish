# Wish UI Kit

Добро пожаловать в UI Kit проекта **Wish** — библиотеку переиспользуемых React-компонентов.

## Структура

| Категория    | Описание                                        |
| ------------ | ----------------------------------------------- |
| **Tokens**   | Дизайн-токены: цвета, шрифты, отступы           |
| **Assets**   | Иконки и другие статические ресурсы             |
| **Core**     | Базовые компоненты (Icon, Button и т.д.)        |
| **Elements** | Простые UI-элементы                              |
| **Molecules**| Составные компоненты из нескольких элементов     |
| **Organisms**| Крупные блоки интерфейса (Header, Footer и др.) |

## Быстрый старт

```bash
pnpm add @wish/ui-kit
```

```tsx
import { Icon } from '@wish/ui-kit';

export const App = () => (
  <Icon category="actions" name="copy" width={24} height={24} />
);
```

## Документация в MDX

Каждый компонент может иметь собственную MDX-страницу рядом со stories-файлом.
Это позволяет писать развёрнутую документацию с примерами, гайдлайнами и живыми превью.
