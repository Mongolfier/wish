# Icon

The `Icon` component loads SVG icons dynamically by name and category.

## Usage example

<Canvas of={IconStories.ExampleIcon} />

## Props

<Controls of={IconStories.ExampleIcon} />

## How it works

Icons are stored in `src/assets/icons/` and grouped by category (folders).
The component uses dynamic `import()` to load the required SVG.

```tsx
<Icon
	category="actions"
	name="copy"
	width={24}
	height={24}
/>
```

If an icon is not found, the component renders an empty `<svg>` and logs an error to the console.
